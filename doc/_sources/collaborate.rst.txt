Collaborate
===========

**Contributions** are welcome! Here's how to get started:

1. **Fork the repository**:

.. code-block:: bash

    git clone https://github.com/APMaii/pygamlab.git
    cd pygamlab

2. **Create a feature branch**:

.. code-block:: bash

    git checkout -b feature/my-feature

3. **Make your changes** and **commit**:

.. code-block:: bash

    git commit -am "Add some feature"

4. **Push to your branch**:

.. code-block:: bash

    git push origin feature/my-feature

5. **Open a Pull Request** on GitHub.

Please follow **PEP8 guidelines**, write **clear docstrings**, and update **tests** when applicable.  
Join our [GAMLab community](https://github.com/APMaii/pygamlab) for discussions and questions.
